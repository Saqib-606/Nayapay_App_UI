# 📱 NayaPay App Clone (Flutter)

A modern NayaPay UI Clone built using Flutter. This project replicates the core design and layout of the NayaPay mobile application.

---

## 🚀 Features

- 🔍 Search bar with user input support  
- 👤 Profile avatar & top navigation icons  
- 💰 Balance display with visibility toggle  
- 🔄 Last updated indicator  
- ⚡ Quick actions (Add Money, Send Money, More)  
- 🎯 Promotional card section  
- 📦 Services grid  
- 📱 Bottom navigation bar  

---

## 🛠️ Built With

- Flutter (Dart)  
- Material Design  
- Responsive Layout (Row, Column, Expanded)
