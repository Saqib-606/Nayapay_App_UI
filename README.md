NayaPay App Clone (Flutter)
A modern NayaPay UI Clone built using Flutter. This project replicates the core design and layout of the NayaPay mobile application, focusing on clean UI, responsive layout, and reusable components.


🚀 Features
🔍 Search bar with user input support
👤 Profile avatar & top navigation icons
💰 Balance display with visibility toggle icon
🔄 Last updated indicator
⚡ Quick action buttons (Add Money, Send Money, More)
🎯 Promotional card section
📦 Services grid:
    QuickPay
    Bills
    Merchants
    Mobile Top-Up
    Bill Split
    Gift Envelope
    Bottom navigation bar with icons


🛠️ Built With
Flutter (Dart)
Material Design Components
Responsive Layout using:
    Expanded
    Row & Column
    Padding & SizedBox


📂 Project Structure
lib/
 └── main.dart        # Main UI implementation

assets/
 └── images/          # App images (icons, assets)


⚙️ Getting Started
Prerequisites
    Flutter SDK installed
    Android Studio / VS Code
    sEmulator or physical device
